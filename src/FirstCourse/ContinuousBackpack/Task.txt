Задача на программирование: непрерывный рюкзак

Первая строка содержит количество предметов 1 <= n <= 10^3 и вместимость
рюкзака 0 <= W <= 2 * 10^6. Каждая из следующих n строк задаёт
стоимость 0 <= c_i <= 2 * 10^6 и объём 0 <= w_i <= 2 * 10^6
предмета (n, W, c_i, w_i — целые числа).
Выведите максимальную стоимость частей предметов (от каждого предмета
можно отделить любую часть, стоимость и объём при этом пропорционально
уменьшатся), помещающихся в данный рюкзак, с точностью не менее трёх знаков после запятой
